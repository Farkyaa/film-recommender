 🎥 Киноподборщик

Интерактивный веб-сайт для подбора фильмов и сериалов с использованием API The Movie Database (TMDb).

 📋 О проекте

Киноподборщик — это современное веб-приложение, которое помогает пользователям находить фильмы и сериалы на основе выбранных жанров, рейтинга, года выпуска, страны производства и других критериев. Проект разработан с использованием React, Next.js и Tailwind CSS.

 Основные возможности:
- 🔍 Поиск фильмов по названию
- 🎯 Фильтрация по жанрам (множественный выбор)
- 📅 Фильтрация по году выпуска и десятилетиям
- 🌍 Фильтрация по стране производства
- ⭐ Сортировка по рейтингу, популярности и похожести
- 🎲 Случайный выбор фильма
- 🔄 Поиск похожих фильмов
- 📱 Детальная информация в модальном окне
- 🔗 Ссылки на Кинопоиск, IMDb, YouTube и другие платформы
- 📊 Нумерация результатов по степени похожести

 🚀 Технологии

- React 18 — библиотека для построения пользовательских интерфейсов
- Next.js 14 — фреймворк для React с поддержкой серверного рендеринга
- TypeScript— строгая типизация для надежного кода
- Tailwind CSS — утилитарный CSS-фреймворк
- TMDb API — внешний источник данных о фильмах
